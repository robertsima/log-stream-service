Welcome to the PrairieLog API

Prarie Dog is a free to use tool for developers to help monitor small applications and services.

The PrairieLog API provides a powerful and flexible way to manage log streams for your applications. With our API, you can easily create, update, and delete log streams, as well as retrieve log stream data in real-time.

PrarieLog generates a unique ingestion token for each application that you register. You can use this token to send JSON formatted log events to the PrairieLog API webhook. Logs sent to the webhook are relayed to the PrairieLog API and then sent to the configured alert destinations such as Slack or Discord.

PrarieLog does not store any logs outside of the alerting system. All logs are processed in real-time for aggregation then forwarded to your configured destinations.
